## Two Types of `Export/Import`

#### You can export `Multiple thing` to use - `Named` Export/Import

### 1. `Default` Export/Import

- export default Component;
- import Component from "path";

### 2. `Named` Export/Import

- export const Component;
- import {Component} from "path";
